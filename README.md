# mk-blogpoetry

This project is a website build using GoogleCloud Platform Python

# Technologies
  1. Python
  2. App Engine
  3. Html/css

# SetUp
  1. Install Python
  2. Install Google App Engine Launcher
  
# Run Project
  1. Clone this repo
  2. <code>cd</code> to this repo
  3. Run command <code>dev_appserver.py</code>
